# Gesit Network - Landing Page & ISP Tools 🌐

Repositori ini berisi **Landing Page Resmi Gesit Network** yang diintegrasikan dengan berbagai alat bantu (tools) teknik jaringan untuk memudahkan teknisi ISP dalam pekerjaan sehari-hari.

---

## 🚀 Fitur Utama

### 1. Landing Page Profesional
- **Desain Modern:** Menggunakan Bootstrap 5 dengan estetika bersih dan premium.
- **Responsif:** Tampilan optimal di perangkat mobile, tablet, maupun desktop.
- **Navigasi Cepat:** Integrasi langsung ke layanan Billing dan sistem Support.

### 2. ISP Tools Hub 🛠️
Kumpulan alat bantu kalkulasi dan generator script:
- **GPON ONU Script Generator:** 
  - Mendukung **ZTE OLT (C320/C300/C600)**.
  - Fitur **Batch Mode** untuk registrasi banyak ONU sekaligus.
  - Auto-generate script hapus (delete) dan konfigurasi (PPPoE/Bridge).
- **Kalkulator Burst Limit MikroTik:** 
  - Menghitung parameter Burst Limit, Threshold, dan Time secara akurat.
  - Langsung menghasilkan script CLI MikroTik yang siap pakai.
- **Kalkulator Rasio Fiber Optic:** 
  - Membantu menghitung redaman dan rasio splitter pada jaringan FTTH.

- **HTML5 & Semantic Tagging:** Untuk struktur SEO-friendly.

## ⚙️ Instalasi & Penggunaan

Proyek ini bersifat **Static Web**, sehingga tidak memerlukan server backend khusus (kecuali jika diintegrasikan dengan API Billing).

1. Clone repositori:
   ```bash
   git clone https://github.com/alfinns99/landing-page.git
   ```
2. Buka `index.html` di browser favorit Anda.
3. Untuk menggunakan tools, navigasikan melalui menu **Tools** pada Navbar.
